## Animation Sequence

### Initial Load
```
1. Page loads normally
2. Blur effect applied to background (0.3s fade)
3. White container slides in from bottom (0.4s)
4. Content appears inside (0.3s fade)
5. Pagination dots animate in (0.2s stagger)
```

### Step Transition (Next Button)
```
1. Content slides left (0.4s)  ← Current step exits
2. Opacity fades out (parallel)
3. New content appears from right (0.4s)
4. Opacity fades in (parallel)
5. Pagination dot transitions (0.3s)
```

### Hover Effects
```
Primary Button Hover:
- Background darkens slightly
- Button lifts up -2px
- Shadow enhances
- Color shift smooth

Secondary Button Hover:
- Background changes
- Border brightens
- Smooth transition 0.3s

Pagination Dot Hover:
- Changes color on hover
- No size change
- Visual feedback
```

```

├─ Onboarding auto-shows
├─ User sees Step 1: Daily Power Check
├─ User clicks "Next"
├─ User sees Step 2: Learning Courses
├─ User clicks "Next"

├─ User clicks "Next"
├─ User sees Step 4: Share Your Story
├─ User clicks "Next"
├─ User sees Step 5: Track Progress
├─ User clicks "Get Started"
├─ Onboarding closes (smooth fade)
└─ localStorage marked as completed

VISIT 2 - Returning User
├─ Page loads
├─ Component checks localStorage
├─ Onboarding is hidden
└─ User goes straight to app
```
